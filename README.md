**Tech Stack:** Python, Flask, HTML, CSS, Jinja2, Open-Meteo API

---

## Description

A modern, responsive web app to get **current weather** for any city.  
Shows **temperature, wind speed, and condition** with a futuristic weather card.  
Includes **About & Contact pages** and **loading animation**.

---

## Installation

1. Clone the repo: `git clone <your-repo-link>`  
2. Create and activate virtualenv  
3. Install dependencies: `pip install -r requirements.txt`  
4. Run app: `python app.py`  
5. Open in browser: `http://127.0.0.1:5000`
