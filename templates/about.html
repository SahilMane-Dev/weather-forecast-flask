About This Project

This Weather Forecast App is built using Flask (Python) and the Open-Meteo API. It provides real-time temperature, wind speed, and weather conditions for any city worldwide.

Technologies Used:

  • Flask Routing
  • Jinja Templates
  • API Integration
  • Responsive & Futuristic UI design

This project is a part of my journey to become a Full Stack Python Developer by building real-world web projects.
